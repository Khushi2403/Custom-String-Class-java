# Custom String Class in Java
This project implements a custom UserString class that mimics important
functionalities of Java's String class.

## Features
- Constructors using String, StringBuilder, StringBuffer, char[], byte[]
- length(), charAt(), concat()
- equals(), equalsIgnoreCase()
- substring(), replace(), trim()
- toUpperCase(), toLowerCase()
- indexOf(), lastIndexOf()
- startsWith(), endsWith()
- valueOf() methods
